## How to run

### Option 1: Run online (easiest, no installation)

1. Open https://www.programiz.com/python-programming/online-compiler
2. Open this repository's file: `legal_engineer_demo.py`
3. Copy the entire code
4. Paste it into Programiz
5. Click **Run**
